iOS上传图片，PHP做为服务器

iOS开发中经常会碰到上传图片的情况，在上传前，先将图片转换成NSData类型，然后利用Post请求，直接将图片放在上传参数即可。
代码如下：

NSData *data = UIImageJPEGRepresentation(self.reasonPic.image, 0.3);
NSMutableDictionary *dic = [NSMutableDictionary dictionaryWithObjectsAndKeys:data,@"pic",str,@"name" ,nil];
AFHTTPSessionManager *manager = [AFHTTPSessionManager manager];
manager.responseSerializer.acceptableContentTypes = [NSSet setWithObjects:@"application/json",@"text/json", @"text/plain", @"text/html", nil];
manager.securityPolicy.allowInvalidCertificates = YES;
[manager POST:@"http://api.zhangchunlin.me/updata.php" parameters:dic constructingBodyWithBlock:^(id<AFMultipartFormData>  _Nonnull formData) {
} progress:^(NSProgress * _Nonnull uploadProgress) {

} success:^(NSURLSessionDataTask * _Nonnull task, id  _Nullable responseObject) {
    if ([[responseObject objectForKey:@"result"] isEqual:[NSNumber numberWithInteger:1]]) {
        self.picURL = [responseObject objectForKey:@"url"];
    }
    NSLog(@"%@", self.picURL);   
} failure:^(NSURLSessionDataTask * _Nullable task, NSError * _Nonnull error) {
    NSLog(@"%@",error);
}];

服务器端，由于PHP是我自学的，当时也是踩了不少坑。当时还遇到过要把NSData类型的16进制转化成2进制，再写入文件的。但是这个方法并不好使。其实直接把接收到的NSData写入指定路径即可。
代码如下：

<?php
$name = $_POST['name'];
$newFilePath="pic/$name.jpg"; //这里如果不是放在本目录下的话，就得给相应的文件夹打开权限，要不无法保存成功
$data = $_POST['pic'];
if(empty($data)){ //判断是否为空
 $code = array('result'=>0, 'url'=>'error');
  $data=file_get_contents($newFilePath);
}  else
{
      $code = array('result'=>1, 'url'=>$newFilePath);
}
$newFile = fopen($newFilePath,"w");
fwrite($newFile,$data);
fclose($newFile);
$json=json_encode($code);
echo $json;  //这里我是将上传后的图片路径返回给客户端
?>
